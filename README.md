# Instagram Clone

An Instagram Clone application developed using Flutter that replicates the core features of Instagram such as posting images, liking posts, viewing stories, and interacting with users through a modern and responsive user interface.

## Features

- User authentication
- Home feed with posts
- Like and comment functionality
- Upload photos and captions
- User profile screen
- Stories section
- Responsive and modern UI
- Real-time updates
- Smooth navigation

## Technologies Used

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Storage

## Functionalities

### User Authentication
Users can create accounts, log in securely, and manage their profiles.

### Post Upload
Users can upload images with captions and share them with others.

### Like and Comment System
Users can like posts and interact through comments.

### Stories Feature
Users can view story updates similar to Instagram.

### User Profile
Each user has a profile page displaying:
- Profile picture
- Bio
- Posts
- Followers and following information

### Responsive Design
The application is designed with a clean and user-friendly interface that works smoothly across different screen sizes.

## Project Goal

The main objective of this project is to develop a social media application using Flutter and Firebase while improving frontend and backend integration skills.
